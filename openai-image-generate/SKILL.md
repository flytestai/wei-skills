# openai-image-generate

当用户要求根据自然语言描述生成图片，并希望直接收到图片文件时，使用本 skill。

## 能力范围
- 接收自然语言提示词
- 调用本地私有配置中的 OpenAI 兼容图片接口
- 生成图片到本地输出目录
- 将图片结果直接发送给当前用户
- 默认不发送链接

## 默认触发规则
- 当用户提出“生成图片 / 画图 / 根据提示词出图 / 帮我做一张图”等请求时，默认优先使用本 skill
- 默认走已配置的 OpenAI 兼容图片接口
- 默认追求接口可接受的最高质量输出
- 生成结果默认直接发送本地图片文件，不发送链接

## 工作流
1. 提取用户提示词，以及可选参数（如尺寸、风格、张数）
2. 调用 `scripts/generate_image.py`
3. 读取脚本输出的本地图片路径
4. 直接发送该本地图片文件给当前用户
5. 如果首次请求超时，可自动做一次更保守参数重试

## 参数约定
- 默认模型：从本地私有配置读取
- 默认尺寸：`1024x1024`
- 默认数量：`1`
- 默认质量策略：优先尝试接口可接受的最高质量参数
- 若用户未提供风格参数，则只传自然语言提示词

## 注意事项
- 不在 skill 文档或脚本中硬编码敏感信息
- 若本地私有配置缺失，先提示配置缺失
- 输出时优先发送本地文件，不发送外链

## 脚本
使用：`scripts/generate_image.py`

如果需要查看接口请求细节或返回结构说明，再读取 `references/api_usage.md`。
