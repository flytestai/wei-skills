# 数据库面试题

> 独立模块：数据库面试题
> 使用规则：与其他模块完全隔离，单独记录进度、错题、收藏、自建题。

## 数据库理论基础面试题

### 1. 什么是数据库、DBMS 和 SQL？它们之间有什么关系
- 关键词：Database, DBMS, SQL
- 说明：理解数据库基础概念和三者关系。

### 2. SQL 语言主要分为哪几类
- 关键词：DDL, DML, DCL, TCL
- 说明：掌握 SQL 语言分类及典型命令。

### 3. CHAR 和 VARCHAR 数据类型有什么区别
- 关键词：CHAR, VARCHAR, 定长, 变长
- 说明：理解存储方式、空间和性能差异。

### 4. DELETE、TRUNCATE 和 DROP 有什么区别
- 关键词：DELETE, TRUNCATE, DROP
- 说明：区分范围、事务、回滚和自增行为。

### 5. WHERE 和 HAVING 子句的区别是什么
- 关键词：WHERE, HAVING, GROUP BY
- 说明：理解过滤时机和聚合函数使用场景。

### 6. INNER JOIN 和 OUTER JOIN 的区别
- 关键词：INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN
- 说明：掌握连接结果差异。

### 7. 如何查询表中存在重复值的记录
- 关键词：GROUP BY, HAVING, COUNT
- 说明：掌握重复值排查 SQL 思路。

### 8. 如何从数据库中随机获取 N 条记录
- 关键词：RAND, NEWID, DBMS_RANDOM
- 说明：了解不同数据库随机取数方式。

### 9. 什么是子查询？有哪些类型
- 关键词：子查询, 标量子查询, 行子查询, 表子查询
- 说明：掌握子查询分类和使用位置。

### 10. UNION 和 UNION ALL 有什么区别
- 关键词：UNION, UNION ALL, 去重
- 说明：理解性能和结果集差异。

### 11. 什么是数据库事务？它的 ACID 属性是什么
- 关键词：事务, ACID
- 说明：掌握事务核心特性。

### 12. 事务的隔离级别有哪些？分别解决了哪些并发问题
- 关键词：读未提交, 读已提交, 可重复读, 串行化
- 说明：理解隔离级别与并发异常关系。

### 13. 什么是脏读、不可重复读和幻读
- 关键词：脏读, 不可重复读, 幻读
- 说明：掌握三类经典并发问题。

### 14. 什么是索引？它的作用和优缺点是什么
- 关键词：索引, 查询优化
- 说明：理解索引价值与代价。

### 15. 哪些字段适合创建索引
- 关键词：主键, 外键, WHERE, JOIN, ORDER BY
- 说明：掌握建索引常见场景。

### 16. 索引是不是越多越好？为什么
- 关键词：索引维护, 空间开销, DML 性能
- 说明：理解索引需要平衡读写成本。

### 17. 什么是聚簇索引和非聚簇索引
- 关键词：聚簇索引, 非聚簇索引
- 说明：掌握物理存储与逻辑索引区别。

### 18. 如何判断一条 SQL 查询是否使用了索引
- 关键词：EXPLAIN, 执行计划, key, type
- 说明：学会看执行计划的基本指标。

### 19. 什么是数据库范式
- 关键词：1NF, 2NF, 3NF
- 说明：理解数据库设计规范。

### 20. 什么是数据库的死锁？如何避免
- 关键词：死锁, 事务, 重试
- 说明：掌握死锁成因和规避思路。

### 21. 主键和外键的作用是什么
- 关键词：主键, 外键, 参照完整性
- 说明：理解唯一标识与表关联。

### 22. 什么是存储过程？它有什么优缺点
- 关键词：存储过程, 预编译
- 说明：掌握存储过程适用场景和局限。

### 23. 什么是触发器？它的作用是什么
- 关键词：触发器, 审计, 自动执行
- 说明：理解触发器触发机制和用途。

### 24. 从测试角度，如何测试存储过程和触发器
- 关键词：数据库测试, 存储过程测试, 触发器测试
- 说明：掌握数据库对象测试思路。

### 25. 如何备份和恢复数据库
- 关键词：完全备份, 差异备份, 日志备份, 恢复
- 说明：理解数据库备份恢复的基本方式。
